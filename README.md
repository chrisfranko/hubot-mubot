# hubot-mubot

Mubot, A Marking U Bot.

See [`src/mubot.coffee`](src/mubot.coffee) for full documentation.

## Installation

In hubot project repo, run:

`npm install hubot-mubot --save`

Then add **hubot-mubot** to your `external-scripts.json`:

```json
["hubot-mubot"]
```

## Sample Interaction

```
user1>> hubot hello
hubot>> hello!
```
